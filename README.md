# Aiogram bot skeleton

<p>Skeleton aiogram bot with admin panel on Postgresql</p>
Stack:
<ul>
    <li>
        Flask-admin
    </li>
    <li>
        SQLAlchemy
    </li>
    <li>
        Aiogram
    </li>
    <li>
        Dependency-injector
    </li>
    <li>
        Postgresql
    </li>
    <li>
        Nginx
    </li>
</ul>

### To start:
```docker-compose up --build```
